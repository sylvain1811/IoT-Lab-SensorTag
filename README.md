# IoT-Lab-SensorTag

SensorTag lab for IoT course at HES-SO Master

## Prerequisites

Hardware:

- TI SensorTag
- Bluetooth adapter

Sorftware:

- Python2.7
- bluepy (`pip install bluepy`)
- Flask (`pip install flask`)

## Configuration

In `backend.py` file, update the sensors_macaddr dict to match your SensorTag Mac adresses.

## Run

```bash
python2.7 app.py
```
